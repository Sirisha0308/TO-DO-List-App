# ✨ To-Do List App

A simple, responsive **To-Do List App** built with **HTML, CSS, and JavaScript**.

## 🚀 Features
- ➕ Add new tasks  
- ✔️ Mark tasks as completed  
- ❌ Delete tasks  
- 🔎 Filter tasks: **All / Active / Completed**  
- 🧹 Clear all completed tasks  
- 📱 Responsive and clean design
